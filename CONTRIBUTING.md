# Contributing to Garden Planner

Thanks for your interest in contributing.

## Getting Started

1. Fork or clone the repository.
2. Install dependencies with `npm install`.
3. Run the app locally with `npm run dev`.

## Development Notes

- Keep changes focused and small when possible.
- Update or add docs when behavior changes.
- If you add new functionality, include basic validation steps in your PR.

## Pull Requests

- Use clear commit messages describing why the change is needed.
- Open a pull request with:
  - a short summary of the change
  - steps to test it
  - screenshots for UI changes

## Reporting Issues

Please use the issue templates and include enough detail to reproduce the problem.
